Conhecimentos

HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language), é uma linguagem de marcação utilizada na contrução de paginas Web.

CSS

CSS, é um mecanimos para adicionar estilos (cores, fontes, espaçamento, etc.) a um documento web. É utilizando essa ferramenta que cosegue-se obter um design agradável e intuitivo.

Linguagem C

A Linguagem C foi a linguagem utilizada na matéria de algoritmos do curso, sendo assim, foi adquirido um bom conhecimento dessa linguagem, sendo ela fundamental para aprendizado de outras linguagens.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Git e GitHub
